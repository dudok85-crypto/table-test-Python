# table-test-Python
Создание простых таблиц на Python с использованием PrettyTable. Настройка отображения таблицы, выравнивание столбцов, добавление заголовков. Импорт данных из CSV-файлов, HTML кода. 

# Необходимые библиотеки для работы проекта: 
1) prettytable
   для установки через терминал или командную строку используйте: pip install prettytable
